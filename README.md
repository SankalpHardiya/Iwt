# Internet and Web Technology Mini Project

It contains a simple static website created as part of the Internet and Web Technology (IWT) course. The project is hosted using GitHub Pages at:  
https://sankalphardiya.github.io/Iwt/

## Features

- Responsive static web pages built with HTML, CSS, Js 
- Clean layout for demonstrating basic web design principles  
- Simple navigation between different sections/pages  
- Deployed using GitHub Pages for easy hosting and access

## Technologies Used

- HTML5  
- CSS3
- Js
- GitHub Pages

## How to View

You can directly visit the live site here:  
https://sankalphardiya.github.io/Iwt/
